## ¿De qué se trata?

Este comparador muestra el **precio actual** de productos Pokémon TCG en diferentes tiendas de Chile, comparado con el **MSRP oficial**.

El objetivo es ayudar a la comunidad a identificar sobreprecios y tomar decisiones más informadas.

---

## Cómo reportar precios

Las Issues y Pull Requests están **desactivadas** para evitar manipulaciones por parte de tiendas.

Si quieres reportar un precio actualizado o una tienda nueva:

- Escríbeme directamente en **X / Twitter**: [@JustSharksimus](https://x.com/JustSharksimus)
- O envíame un correo a: `arctikrensen@gmail.com`

Solo actualizaré con información verificada.
